📘 SQL Problem Statements & Solutions
This repository contains a comprehensive set of SQL problem statements along with their solutions. It covers a wide range of SQL concepts including:

✅ Basic Queries with WHERE, LIKE, DISTINCT, CASE

📊 Aggregations, GROUP BY, HAVING, Date Functions

🔗 Joins, Subqueries, and Self-Joins

📦 Constraints, DDL Commands, Views

🛠️ Stored Procedures, Triggers, and Error Handling

📈 Window Functions and Real-world Reporting Scenarios

Each query is written and tested using the classicmodels sample database and other custom-created tables.

Great resource for practice, interview preparation, and SQL mastery.
